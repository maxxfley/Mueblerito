# Modelo de negocio

## Ubicaciones

- Local o sucursal X

## Problemas actuales

- No existe control centralizado del stock.
- Los vendedores deben llamar a otras sucursales para consultar disponibilidad.
- Se venden muebles sin informar a los demás sectores de forma eficiente.
- Los pedidos a medida se mezclan o retrasan.

## Objetivo

Centralizar la información de muebles, stock y pedidos.
